1 Maxwell Theory

1.1 Integral form

We are going to describe the Maxwell equations in integral form. For this, we need a bunch of definitions. Let

  • \(U ⊂ 𝐑³\) be a 3-dimensional region in 3-space

  • \(dv\) denote the volume element on \(U\)

  • \(∂U\) denote the (2-dimensional) boundary of \(U\)

  • \(\bf n\) denote the (outward-directed) unit normal to \(∂U\)

  • \(da\) denote the area element on \(∂U\) induced from the canonical volume form on \(𝐑³\) and let \(d{\bf a} = {\bf n} da\).

  • \(A⊂ 𝐑³\) denote an oriented surface

  • \(d\bf l\) denote the line element on \(∂A\) induced from the canonical volume form on \(𝐑³\)

The Maxwell equations are

Coulomb

Let \(Q(U)\) be the charge contained in \(U\). Then \[%Q(U) = \frac 1{4 π } ∫_{∂U} {\bf E} ∙ {\bf n}\ da 4π Q(U) = ∫_{∂U} {\bf E} ∙ d{\bf a}\]

No monopoles

There are no magnetic sources: \[0 = ∫_{∂U} {\bf B} ∙ d{\bf a}\]

Ampere

Let \(\bf j (A)\) denote the current (charge flux) flowing through the surface \(A\). Then \[∫_{∂A} {\bf B} ∙ d{\bf l} = \frac{4π}c {\bf J}(A) + \frac 1c {∂ \over ∂ t} ∫_A {\bf E}∙ {\bf n}\ da\]

Faraday-Lenz

\[∫_{∂A} {\bf E} ∙d{\bf l} = - \frac 1c {∂ \over ∂ t} ∫_A {\bf B}∙ {\bf n}\ da\]

1.2 Traditional Calculus 3 form

Let

  • \(f\) and \(g\) denote functions \(𝐑³ → 𝐑\)

  • \(\bf u\) and \(\bf v\) denote vector fields on \(𝐑³\)

  • \((x,y,z)=(x¹, x², x³) = (x^i)_{i=1}³\) denote Cartesian coordinate functions on \(𝐑³\)

In calculus 3, we learn about the differential vector operators

coordinate basis

\(❴ ∂_i ❵ _{i=1}³\) with \(∂_i = \frac ∂ {∂x^i}\). By definition of Cartesian coordinates, the coordinate vectors commute \(∂_i ∂_j = ∂_j ∂_i\).

gradient

The gradient \(\textrm{grad}(f)\) or \(∇f\) of a function is a vector field with components \[∇f = (∂^i f)= (δ^{ij} ∂_j f) = \ma{ \frac {∂f} {∂x} \\ \frac {∂f} {∂y} \\ \frac {∂f} {∂z} }\]

divergence

The divergence \(\mathrm{div} ({\bf u})\) or \(∇ ∙ {\bf u}\) of a vector field is a scalar defined by \[∇ ∙ {\bf u} = \frac {∂ u¹} {∂x} + \frac {∂ u²} {∂y} +\frac {∂ u³} {∂z} = ∂_i u^i\] Here and henceforth, repeated indices with one “up” and one “down” are summed (Einstein summation convention).

curl

The curl \(\mathrm{curl}({\bf u})\) or \(∇ × {\bf u}\) of a vector field is a vector field with components defined by \[(∇ × {\bf u})³ = \frac {∂ u²} {∂x¹} - \frac {∂ u¹} {∂x²}\] and cyclic permutations of \(¹,²,³\).

We also learn that these operators are related by integration. In particular, for subsets of \(𝐑³\) in the notation of the previous section, we have the following theorems:

FTC?

Let \(f\) be a function on (a neighborhood of) of an oriented curve \(γ\) with boundary \(∂γ = b - a\), possibly with \(b=a\) (no boundary).1 Then \[∫_γ (∇ f) = ∫_{∂γ} f = f|^b_a = f(b) - f(a)\]

Green

Let \(𝒖\) denote a vector field on (a neighborhood of) \(A\). Then \[∫_A (∇ × 𝒖) ∙ d 𝒂 = ∫_{∂A} 𝒖 ∙ d𝒍\]

Stokes

Let \(𝒖\) denote a vector field on (a neighborhood of) \(U\). Then \[∫_U (∇ ∙ 𝒖)\ dv = ∫_{∂U} 𝒖 ∙ d 𝒂\]

Ever feel like Green’s theorem and Stokes’ theorem were secretly the same thing?

We now define the charge density \(ρ\) and the current density \(𝒋\) so that \[Q(U) = ∫_U ρ dv \quad\quad \textrm{and}\quad \quad {\bf J}(U) = ∫_U 𝒋 dv\]

1.3 Superior mathematical form

The differential “calc 3” notation is clearly superior to the integral form, it is still dirty for various reasons.

Exercise 3. Restrict the entire universe to two dimensions instead of 3. What does the cross product mean in this 2D universe? Extend to a 4D universe by adding an additional direction to up/down, left/right, forward/backward. Can you define the cross product there?

The problem is that the existence of the cross product is an accident of 3D. A cross product is an anti-symmetric pairing to two vectors/vector fields to give a third vector (field) and this does not need to exist in arbitrary dimension.

We will return to the reason it exists in 3D eventually, but for now, let’s see how we can salvage the Maxwell equations by stripping off the 3D-ness. To do this, we need some formalism.

Definition 1. Let \(V\) be a real vector space. A linear form on \(V\) is a linear map \(V → 𝐑\). The space of linear forms on \(V\) is denoted by \(V*\); it is called the dual of \(V\).

Claim 1. Let \(\textrm{dim}_𝐑 (V) < ∞\). Then \(V*\) is a vector space and \(\textrm{dim}_𝐑 (V*) = \textrm{dim}_𝐑 (V)\).

Definition 2. Let \(𝒖\) be a vector field on \(U⊂ 𝐑³\) and let \(❴x^i ❵_{i=1}³\) be a set of coordinates on \(U\). In these coordinates \[𝒖 = u^i(x) ∂_i = u^i(x) \frac ∂{∂x^i}\] For the functions \(u^i :U→ 𝐑\) for \(i=1,2,3\) are called the components of \(𝒖\) (in the coordinates \(x\)).

Take careful note of the position of the indices in the expressions above: The the component functions of vector fields have their indices “up”, whereas the coordinate basis vectors \(∂\) have their indices “down”.

Now suppose we have the component functions \(❴u^i❵_{i=1}³\) for some vector field \(𝒖\). I’m getting tired of typing that out, so when the meaning is clear, I will abbreviate this as \(❴u^i❵\). Anyway, to map from a set like this to a real function, we need to “get rid of the index”: If we have 3 other functions \(❴ω_i(x)❵\), we can make \[ω(𝒖) = ω_i(x) u^i(x)\] which is a function \(U→ 𝐑\), as we wanted when we defined linear forms.

Our conclusion is that under the conditions set out in the Claim, the dual space is spanned by elements with coordinate functions \(❴ω_i❵\) analogous to the \(❴u^i❵\) of a vector field, but with the indices in the other position. There is a good reason for this, but for the moment you can just take it as a rule.

If you don’t want to hit the “I believe” button, consider the following: Let \(L : V→ V\) denote a linear operator on a vector space \(V\), and consider its action on a vector \(𝒖 ↦ L(𝒖)\). This operator does not act on \(𝐑\) so if we consider a linear form \(ω : V→ 𝐑\) then it maps \(𝒖 ↦ ω(𝒖) ∈ 𝐑\). This implies that \(L\) must act on \(ω\) in a way that ensures that it has no effect on \(ω(𝒖)\) (since it is an element of \(𝐑\) and \(L\) has no effect there.

A linear form \(V→ 𝐑\) is also called a 1-form. When we have a coordinate basis for \(V\) denoted by \(❴∂_i= ∂/∂x^i❵\) , then there is a natural basis for \(V*\) denoted \(❴dx^i❵\). (Note the position of the index.) Since 1-forms map vectors to numbers, it must be true that the \(dx\)s map the \(∂\)s to numbers. The definition/normalization for Cartesian coordinates is this:3 \[dx^i ( ∂_j ) = δ^i_j = \begin{cases} 1 & \textrm{if } i=j\\ 0 & \textrm{otherwise} \end{cases}\]

Exercise 5. We can combine all theses things into the following synthesis: Let \(𝒖\) be a vector field and let \(ω\) be a 1-form field, both defined on \(U⊂ 𝐑^n\) for general \(n\). Let \(❴x^i❵\) denote coordinates on \(U\). In these coordinates, \(𝒖=u^i(x) ∂_i\) and \(ω = ω_i(x) dx^i\) where the repeated indices are summed (as always). Then from these expressions and the definitions show that \[ω(𝒖) = ω_i(x) u^j(x) dx^i(∂_j) = ω_i(x) u^j(x) δ^i_j = ω_i(x) u^i(x)\]

This is an example of a calculation done in a local coordinates called a “component calculation”. We will do many of these.

1.3.0.1 Multilinear algebra

We can make combinations of the basis 1-forms by taking more copies of \(V*\) in what is called the “tensor product”. Operationally, the tensor product \(V*⊗V*\) is just the collection of objects with basis \(dx^i ⊗ dx^j\) where the \(⊗\) symbol roughly means “keep them in this order”. For example, if \(M ∈ V*⊗V*\) then what this means is that in local coordinates \(M = M_{ij} dx^i ⊗ dx^j\). We recognize the components \(M_{ij}\) as those of an \(n× n\) matrix.

We can similarly do this for \(V\) as well and we can mix \(V\) and \(V*\). Furthermore, we can make arbitrarily higher numbers of factors like \(V ⊗ V ⊗ V* ⊗ V* ⊗ V\). Often, we rearrange the factors so that all the vector spaces come first and then all the duals,4 so we could write this as \(V ⊗ V ⊗ V ⊗ V* ⊗ V*\).

An element of \(V ⊗ \cdots ⊗ V ⊗ V* ⊗ \cdots ⊗ V*\) with \(q\) factors of \(V\) and \(p\) factors of \(V*\) is called a tensor of rank \((q,p)\). Tensors of low rank often have special names: A tensor of rank

  • \((0,0)\) is a scalar

  • \((1,0)\) is a vector

  • \((0,1)\) is a linear form or 1-form

  • \((0,2)\) is a bilinear form 2-form

  • \((0,q)\) is a multi-linear form of rank \(q\) or \(q\)-form

  • \((1,1)\) is a linear operator

  • \((q,p)\) with \(p+q=2\) are also called matrices

Permutations have a natural action on tensors, and we use them to reduce the general tensors to simpler elements. For example a \((0,2)\) tensor \(T\) can be split into symmetric \(S\) and anti-symmetric \(A\) parts: \(T_{ij} = S_{ij} + A_{ij}\). This is so useful that we use a special notation on the indices: \[S_{ij} = T_{(ij)} = \frac12 (T_{ij} + T_{ji}) \quad \quad \textrm{and}\quad \quad A_{ij} = T_{[ij]} = \frac12 (T_{ij} - T_{ji})\]

An exterior \(q\)-form is an element \[ω ∈ Λ^q = ⋀_{i=1}^q V*= V*\wedge \cdots \wedge V*\] In local coordinates \(❴x^i❵_{i=1}^n\) on \(U\), an exterior form of rank \(p\) can be written as \[ω = ω_{i₁ ... i_p}(x) dx^{i₁} \wedge \cdots \wedge dx^{i_p} %\quad \quad \textrm{where} \quad \quad %ω_{i₁ ... i_p}(x)\] where the component functions \(ω_{i₁ ... i_p}(x) = ω_{[i₁ ... i_p]}(x)\) are totally anti-symmetric under permutation of indices.

Definition 3. Let \(d_i : Λ^i → Λ^{i+1}\) for \(i=0,\dots,n\) be a set of maps. Such a set of maps is called a differential on \(U\) if \(d_{i+1} ∘ d_i = 0\) for all \(i\). We define an extension of these maps to forms of any degree by defining \(d_i : Λ^j → Λ^{j+1}\) to be the 0 map whenever \(i≠ j\). Finally, using this extension, we define \(d : Λ^p(U) → Λ^{p+1}(U)\) for any \(p\) by \(d = ∑_{i=0}^n d_i\).

Here is an important application/example: For each \(i,p=0,...,n\), let \(ω∈ Λ^p(U)\) and define \[d_i(ω) = \begin{cases} ∂_k ω_{j₁ ... j_i}(x) dx^k \wedge dx^{j₁} \wedge \cdots \wedge dx^{j_i} & \textrm{if } i=p\\ 0 & \textrm{otherwise} \end{cases}\]

This exercise has very interesting consequences for mathematics: An algebra of exterior forms equipped with a differential is called a differential algebra, so the first two conditions are saying that there is such a thing embedded in the structure of calculus. The third part of the exercise is interesting, because you cannot drop the qualifier that \(U\) is an open ball.

A form \(ω ∈ Λ^p(U)\) is called “closed” if \(dω=0\). A form \(ω ∈ Λ^p(U)\) that can be written as \(ω = d λ\) for some \((p-1)\)-form \(λ\) is called “exact”. Note that exact forms are always closed because \(d(dλ) = 0\) by the very construction of the differential. However, closed forms are not always exact.5

Often in physics, quantities that we refer to as “vectors” are actually 1-forms. For example, while it is true that the gradient \(∇f\) of a function is a vector, the more natural object is actually the 1-form \(df\). The two are related with the dot product: A dot product \(\langle ∙ , ∙ \rangle\) is a map \(V⊗V→ 𝐑\) taking two vectors \((𝒖, 𝒗 )↦ \langle 𝒖, 𝒗 \rangle = 𝒖 ∙ 𝒗\). This means that if we fill only the first slot, say, with a vector \(\langle 𝒖 , ∙ \rangle\) then we get a map \(V→ 𝐑\), that is, we get an element of \(V*\), or in yet other words, we get a 1-form. You should confirm that \[df = \langle ∇ f, \cdot \rangle\] In fact, if you think about how we make a gradient, we take the components of \((df)_i = ∂_i f\) and arrange them as a column vector: \[( (df)_x , (df)_y, (df)_z) ) = (∂f/∂x, ∂f/∂y, ∂f/∂z) ⟶ \ma{∂f/∂x\\ ∂f/∂y\\ ∂f/∂z}\] But that means that we are making a vector out of the components of the 1-form!6

1.4 Special relativity

In special relativity, we treat the time coordinate \(t\) on an equal footing with the spacial coordinates \(x^i\). To do this, we define \(x⁰ = ct\) so that it has the correct units. Then we extend the index range \(x^i → x^m\) with \(m=0,1,2,3\). Note that \(∂_i → ∂_m\) with \(∂₀ = \frac1c \frac ∂{∂t}\). The (relativistic) wave operator \(☐\) is defined by \[☐ = -\frac1 {c²} \frac {∂²}{∂t²} + ∇² = -\frac1 {c²} \frac {∂²}{∂t²} + \frac {∂²}{∂x²}+\frac {∂²}{∂y²}+\frac {∂²}{∂z²}\] It is called the d’Alembertian. Notice that it looks like \((∂_m)²\) except that the sign of the temporal component is reversed! For this reason, we define \(∂^m\) to have the components \(∂⁰ = - \frac1c \frac ∂{∂t}\) and \(∂^i = δ^{ij} ∂_j\) (as before).

The 4D inner product defined is defined to have components \[η_{mn} = \begin{cases} -1 & \textrm{ if } m=0=n \\ +1 & \textrm{ if } m=n ≠ 0\\ 0 &\textrm{ otherwise } \end{cases}\] It is called the “Minkowski metric”. The “inverse” metric \(η^{mn}\) is defined to have the same components numerically. This is so that it is, in fact, an inverse: \(η^{mp}η_{pn} = δ^m_n\) for any \(m\) and \(n\) (including 0).

1.5 Integration

Differential forms can be integrated. To integrate a \(1-form\), we need a 1-dimensional subspace of \(𝐑^n\), that is, a curve. To integrate a \(2-form\), we need a surface, and we integrate a 3-form over a volume, et cetera. The curve, surface, and volume on which we integrate are called \(p\)-chains for \(p=1,2,3\). (A 0-chain is an ordered collection of points.) The space of \(p\)-chains is denoted by \(Λ_p\). Integration is a pairing of \(Λ_p\) with \(Λ^p\): We will not go into the details of chains, but you can imagine that a \(p\)-chain can have a boundary that is a \((p-1)\)-chain. We write the boundary of a chain \(γ\) as \(∂γ\).

Exercise 16. Convince yourself that the boundary of a boundary is empty.

In fact, it turns out that the space of chains can also be reformulated to have the structure of a differential algebra where the differential is the boundary operator. For this to be valid, we need the condition that \(∂² = 0\), but that is precisely the statement that the boundary of a boundary is empty. Note that \(∂ : Λ_p → Λ_{p-1}\) so the differential moves us down in the degree instead of up.
